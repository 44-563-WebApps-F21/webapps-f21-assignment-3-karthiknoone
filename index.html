Bipartite Graph Check

A Bipartite graph (or bigraph) is a graph whose vertices can be divided into two disjoint and independent sets. There exists a theorem which claims that a graph is bipartite if and only if all its cycles have even length.A graph is bipartite if and only if it does not contain an odd cycle.A graph is bipartite if and only if it is 2-colorable, (i.e. its chromatic number is less than or equal to 2).Any bipartite graph consisting of ‘n’ vertices can have at most (1/4) x n^2 edges.The spectrum of a graph is symmetric if and only if it is a bipartite graph.
